">" -> move pointer right.
"<" -> move pointer left.
"+" -> increase current memory value by one.
"-" -> decrease current memory value by one.
"." -> push current value in memory to the output stack.
"," -> take one character keyboard input from user.
"[" -> open loop.
"]" -> close loop.
"{" -> open if statement.
"}" -> close if statement.
"~" -> dump memory.
